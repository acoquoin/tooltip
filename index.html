Example

Simple example

This is a simple exemple of implementation:

for (element of document.querySelectorAll('[data-title]')) {
    let instance = new Tooltip(element);
}

Will render like this: just mouseover it !

Advance example

Or, you can do something more complicated, like loadingcontent via a querySelector, and why not, special style: 

for (element of document.querySelectorAll('[data-title]')) {
    let instance = new Tooltip(element, function() {
        if (['#', '.'].indexOf(element.dataset.title[0]) === 0) {
            return document.querySelector(element.dataset.title).innerHTML;
        }
        return element.dataset.title;
    }, 'custom');
}

Adding a little bit of CSS:

#tooltip-custom {
    color: #000;
    background: #FFF;
    border: 1px solid rgba(0, 0, 0, .75);
    border-radius: 5px;
    margin: 0;
    box-shadow: 5px 5px 10px 0 #AAA;
    max-width:500px;
}
#tooltip-custom::after {
    display:none;
}

And a little bit of HTML:

<template id="custom">I'm a content into a <i>template</i> tag, loaded by <strong>querySelector<strong>.</template>

Will render like this: just mouseover it !
